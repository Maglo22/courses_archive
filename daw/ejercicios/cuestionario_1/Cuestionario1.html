Cuestionario 1

¿Qué es una basde de datos?

Conjunto de información relacionada lógicamente que se encuentra agrupada o estructurada.

¿En qué casos es conveniente usar bases de datos?

Cuando se necesita manejar una gran cantidad de datos que se pueden relacionar de alguna manera, de esta manera se pueden automatizar procesos y hacer búsquedas avanzadas de una manera más rápida.

¿Qué es un sistema de gestión de base de datos?

Tipo de software dedicado a servir de interfaz entre la base de datos, el usuario u las aplicaciones que la utilizan.

Funciones/responsabilidades de los DBMS

Definición de los datos - definir todos los objetos de la base de datos partiendo de definiciones fuente.
Manipulación de los datos - realizar operaciones de supresión, actualización, extracción, entre otros.
Seguridad e integridad de los datos - aplicar medidas de seguridad para garantizarla frente a ataques o por simple restricción hacia ciertos usuarios.
Recuperación y restauración de los datos.

Ejemplo de proyecto en el que hubiera sido conveniente utilizar una base de datos

En un proyecto de Probabilidad y Estadística, en el cual se necesitaba describir una gran cantidad de datos en gráficas. Aunque se usó una hoja de Excel para consultar los datos, no todos se encontraban bien recaudados, habían distintos formatos en la misma pregunta, entre otros problemas.
