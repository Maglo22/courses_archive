¿Qué diferencias y semejanzas hay entre Java y JavaScript?

JavaScript no es un lenguaje de programación, y tampoco es compilado. Está basado en objetos mientras qu Java es un lenguaje orientado a objetos. En JS no es necesario la declaración de variables a utilizar. En general JS se ejecuta en tiempo real, mientras que en Java se necesita una compilación previa para realizar comprobaciones previas.

¿Qué métodos tiene el objeto Date?

Entre los métodos con los que cuenta el objeto Date:
getDate() - regresa el día del mes.
getDay() - regresa el día de la semana.
getHour() - regresa la hora.
getUTCDate() - regresa el día del mes según el tiempo universal.
getTime() - regresa el número de milisegundos desde la media noche de Enero de 1970, y una fecha en específico.

¿Qué métodos tienen los arreglos?

Algunos métodos de los Arreglos:
toString() - convierte los elementos de un arreglo en una String, separados por comas.
join() - junta los elementos de un arreglo en una String.
pop() - quita el último elemento de un arreglo.
push() - añade un nuevo elemento al final del arreglo.
shift() - elimina el primer elemento del arreglo y mueve los demás una posición adelante.

¿Cómo se declara una variable con alcance local dentro de una función?

var nombre. Cuando se declara una variable dentro de una función, se vuelve local a la función y sólo puede ser usada por código dentro de la función. Se crean cuando la función se ejecuta y desaparecen cuando es completada.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Puede causar conflicto lógico con otras funciones, o se pueden cometer errores al declarar variables con el mismo nombre dentro de otras funciones, a pesar de que sean locales.

¿Qué método de String se puede utilizar para buscar patrones con expresiones regulares? ¿Para qué podrías utilizar esto en una aplicación web?

El método search() de String. Se puede utilizar para crear una función que le permita a la aplicación realizar búsquedas avanzadas dentro un gran número de datos.
