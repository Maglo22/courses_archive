¿Qué importancia tiene AJAX en el desarrollo de RIA's (Rich Internet Aplications)?
AJAX le permite a una aplicación web ser más parecida a una de escritorio. Las llamadas asíncronas evitan la constante recarga de una página al hacer cambios mínimos y termina dándole al usuario una mejor y más fluida experiencia.
¿Qué implicaciones de seguridad tiene AJAX? ¿Dónde se deben hacer las validaciones de seguridad, del lado del cliente o del lado del servidor?
Al funcionar a base de JavaScript, se necesita estar seguro que no es posible realizar algún tipo de injección de código. Es importante validar todas las peticiones enviadas, si es posible tanto del lado del cliente como del servidor, pero dándole más importancia al servidor, en donde se encuentran todos los recursos almacenados.
¿Qué es JSON?
JavaScript Object Notation. Es una sintáxis de JavaScript orientada a objetos. Sirve para el almacenamiento e intercambio de datos. Un mediador entre el JavaScript de la página y el servidor; convierte a texto el JS cuando se comunica con el servidor, lo traduce a JS cuando el servidor manda la respuesta.
