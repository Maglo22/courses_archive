¿Por qué es buena práctica separar el modelo del controlador?
Para hacer más mantenible el código al tener una mejor organización.
¿Qué es SQL Injection?
Inserción de código SQL no deseado a través de entradas en una página web.
¿Qué técnicas puedes utilizar para evitar ataques de SQL Injection?
Se pueden utilizar parámetros SQL para protección, comprobar que las entradas son del tipo de datos previsto, escapar las variables por medio de funciones como "mysql_real_escape_string()", entre otras.
